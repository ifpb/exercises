# Soma de Inteiros

Crie uma função que ao receber um vetor unidimensional gere uma soma de todos os valores inteiros. Para analisar mais exemplos veja a _Tabela 1_.

_Tabela 1:_

| Entrada                      | Saída |
| ---------------------------- | ----- |
| `()`                         | `0`   |
| `(1, 2, 3, 20)`              | `26`  |
| `((1, 3), (2, 2))`           | `8`   |
| `(2, (3, 4), (3, 1))`        | `13`  |
| `(1, 3, (3, 2, (), (3), ())` | `12`  |
