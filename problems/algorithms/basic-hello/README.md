# Hello World

Crie um algoritmo que exiba uma saudação utilizando o nome de alguma variável. Então, ao definir a string `"Luiz"` o retorno deve ser `Hello, Luiz!`.

Para analisar mais exemplos veja a _Tabela 1_.

_Tabela 1:_

| Entrada | Saída           |
| ------- | --------------- |
| `Luiz`  | `Hello, Luiz!`  |
| `Alice` | `Hello, Alice!` |
| `IFPB`  | `Hello, IFPB!`  |
