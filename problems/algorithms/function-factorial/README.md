# Fatorial

Crie uma função para calcular o fatorial de um valor, por exemplo, se recebermos o valor `4`, então o seu fatorial será `24`. Para analisar mais exemplos veja a _Tabela 1_.

_Tabela 1:_

| Entrada | Saída |
| ------- | ----- |
| `1`     | `1`   |
| `2`     | `2`   |
| `3`     | `6`   |
| `4`     | `24`  |
