# Compras

Crie um sistema de compras baseado nestes itens:

| ID  | Preço | Quantidade |
| --- | ----- | ---------- |
| 1   | 10,00 | 2          |
| 2   | 10,00 | 2          |
| 3   | 10,00 | 2          |
| 4   | 10,00 | 0          |

Então, gere uma função para calcular o valor total do carrinho, que no caso destes itens será R\$ 60,00.
